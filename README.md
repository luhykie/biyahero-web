#  BiyaHero: Ride Saver

BiyaHero is a transportation budget tracking web application that helps users manage commuting expenses, track trips, and apply smart strategies to save money on daily travel.

---

##  Features

###  Commute Setup
- Select month (e.g., April 2026)
- Choose trip type (One-way / Two-way)
- Pick active commuting days
- Automatically tracks total trips

###  Budget Overview
- Total Spent Target
- Budget Left
- Total Saved
- Money Saved per ride

###  Savings Form
- Input current location and destination
- Set monthly transportation budget
- Auto-fill number of trips based on schedule

###  Hero Moves (Cost-Saving Options)
Users can select strategies to reduce expenses:
- Walk short distances
- Use BRT / bus corridors
- Avoid rush hours
- Avoid ride-hailing apps
- Avoid tricycles
- Prefer direct routes
- Apply Student / PWD / Senior discounts

###  Dashboard Stats
- Trips tracked (monthly)
- Trips this month
- Savings insights

###  Trip History
- Stores recorded trips (future-ready feature)

###  Map Legend
- Origin
- Destination
- Terminal
- Landmark stop
- Main route
- Suggested PUJ path

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript, React  
- **Backend:** JavaScriptNode.js 
- **Database:** MySQL   

---



##  How to Use

1. Set your commute schedule and trip type  
2. Enter your monthly transportation budget  
3. Input your origin and destination  
4. Select applicable **Hero Moves**  
5. Click **Track Savings**  
6. View your savings, trips, and budget status  

---

##  Purpose

This system helps:

- Track commuting expenses  
- Encourage smart travel decisions  
- Promote cost-saving habits for daily commuters  

---
